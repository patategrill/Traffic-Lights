# Traffic-Lights in MicroPython for micro:bit cards🚦

For this traffic light, you need 2 micro:bit cards.

### Features ✨ 

- Synchronize cards for traffic ligths
- Wireless communication with radio

### Requirements 🛠
- 2 micro:bit cards
- An editor such as Thonny or MicroPython editor 


### Installation

1. Open the code in Thonny or the micro:bit Python Editor
2. Convert the `.py` file into a `.hex` file
3. Copy the `.hex` file onto each micro:bit
4. Restart the cards

You can run it in code editor such as Thonny.  
To import this code in your microbit card, you must convert your file .py to a .hex file using,for example, [microbit editor](https://python.microbit.org/v/3) and copy your file in the microbit card
